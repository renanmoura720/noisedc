# Mapeamento sistemático

Protocolo, strings de busca, planilha de triagem e diagrama de seleção (Sankey)
referentes à revisão de literatura da dissertação (Capítulo 2).

Inclua aqui:

- `protocolo.md` — questões de pesquisa, critérios de inclusão e exclusão, bases consultadas;
- `strings-de-busca.md` — as strings efetivamente utilizadas em cada base;
- `planilha-triagem.csv` — os estudos triados, com a decisão e a justificativa;
- `diagrama-selecao.png` — o fluxo de seleção dos estudos.

Este material sustenta a fundamentação teórica da Camada 3 e das escolhas de
projeto da Camada 1.
